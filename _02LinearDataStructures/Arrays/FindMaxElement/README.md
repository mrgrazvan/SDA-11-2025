## Problem Description

Given an array of integers `nums`, return the **maximum** value in the array.

---

## Example 1

```
Input: nums = [3, 5, 1, 2, 4]
Output: 5
```

## Example 2

```
Input: nums = [10]
Output: 10
```

---

## Constraints

- `1 <= nums.length <= 10^4`  
- `-10^9 <= nums[i] <= 10^9`
